# Password Generator

A simple command-line password generator written in Python.

This project was built to practice:
- basic Python syntax
- user input handling
- loops and conditions
- the "random" module
- the "string" module
- methods

## Features

- Generate random passwords
- Choose password length
- Uses letters, numbers, and symbols
- security check
- Runs in the terminal

## How it works

The program randomly selects characters from predefined character sets, combines them into a password of the desired length, and evaluates its strength.

## Requirements

- Python 3.x

No external libraries required.

## Usage

Run the script in the terminal:

```bash
python main.py
